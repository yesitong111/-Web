工厂流水线3D可视化展示系统

项目简介
本项目是一个基于Web技术的工厂流水线3D可视化展示系统，旨在通过Three.js技术实现一个高度真实的工业车间场景。用户可以通过浏览器自由观察3D场景、控制设备运行状态，体验真实的工业生产环境。该项目是浙江树人学院Web前端设计与开发课程的期中作业。

功能介绍
一、3D场景展示
系统实现了完整的工厂车间3D场景，包含地面、墙面、支撑柱、横梁、通风管道等厂房结构，以及流水线设备模型。场景采用透视视角，用户可以通过鼠标自由旋转、平移和缩放视角，从任意角度观察车间环境。
二、光照系统
系统采用多层光照架构，包含环境光、方向光、聚光灯和点光源。环境光提供基础照明，方向光模拟自然天光，5盏聚光灯模拟车间顶灯，3盏彩色指示灯模拟设备状态灯。所有光源均启用阴影功能，增强场景真实感。
三、材质与纹理
系统使用程序化纹理技术生成所有材质，避免外部资源依赖。地面采用灰色瓷砖纹理，墙面采用浅灰白色纹理并印有安全标识，金属结构采用银灰色金属纹理，安全区域采用绿色和黄色黑色警戒线纹理。
四、交互动画
系统提供丰富的交互功能。用户可以通过控制面板启动或停止设备运行，调节动画速度，复位视角。设备运行时，传送带呈现滚动效果，设备部件持续旋转，指示灯发出柔和的辉光。点击场景中的对象可以查看相关信息面板。
五、视觉效果
系统使用后期处理技术增强视觉效果，包括辉光效果、色调映射和抗锯齿。辉光效果使指示灯和高光区域产生真实的光晕，ACES色调映射优化画面色彩对比度，提升整体视觉质感。
六、性能优化
系统在多个层面进行了性能优化，包括纹理优化、阴影优化、渲染优化和内存管理。程序化纹理减少加载时间，合理的阴影贴图分辨率保证性能，轨道控制器阻尼效果提升交互体验。

运行环境
浏览器：Chrome、Firefox等现代浏览器，需支持WebGL 2.0
硬件：支持WebGL 2.0的显卡，推荐中等性能以上计算机
分辨率：建议1920x1080及以上
网络：本地运行无需网络连接

使用方法
一、使用VS Code打开项目文件夹
二、安装Live Server插件
三、右键点击index.html文件，选择Open with Live Server
四、在浏览器中访问项目页面

操作说明
视角控制：鼠标左键拖动旋转视角，鼠标右键拖动平移视角，滚轮缩放
动画控制：点击运行按钮启动设备动画，点击停止按钮停止动画，点击重来按钮重置动画状态，点击复位视角按钮恢复初始视角
速度调节：拖动滑块调整动画播放速度，范围从0.1倍到3倍
点击交互：点击指示灯、安全区域或流水线模型可以查看相关信息
